# BTRIX Brain Changelog

All notable changes to the BTRIX Knowledge Brain will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-02

### Added

#### Core Documents
- **BTRIX_CORE.md** - Complete system overview and philosophy
- **BTRIX_PACKS.md** - Service packages, pricing, and bundles
- **BTRIX_AGENTS.md** - AI agents specifications and pricing
- **BTRIX_FAQ.md** - Frequently asked questions
- **BTRIX_LIMITS.md** - Clear boundaries of what BTRIX doesn't do

#### Repository Structure
- Created `/core/` directory for knowledge documents
- Created `/changelog/` directory for version history
- Created `/scripts/` directory for future automation
- Added **README.md** with usage guidelines

#### Versioning
- Established semantic versioning system (MAJOR.MINOR.PATCH)
- Defined change management process (PR-based)

### Specifications

#### Pricing (v1.0.0)
- **Essential:** €1,400 setup + €300/month
- **Pro:** €2,200 setup + €550/month
- **Enterprise:** €3,500+ setup + €900+/month

#### Agents (v1.0.0)
- Sales Agent: €200/month
- Marketing Agent: €200/month
- Finance Agent: €180/month
- Inventory Agent: €180/month
- Social Media Agent: €180/month
- Design Agent (Image): €180/month
- Video Agent: €250/month

#### Bundles (v1.0.0)
- Essential + Agents: €430/month (save €250)
- Pro + Agents: €790/month (save €520)
- Enterprise + Agents: €1,350/month (save €640)

### Philosophy
- Established "Single Source of Truth" principle
- Defined "Automation you don't even notice" as core philosophy
- Set clear boundaries (BTRIX_LIMITS.md)
